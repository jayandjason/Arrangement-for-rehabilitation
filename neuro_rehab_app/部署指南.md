# 神经心理康复中心病人预约管理系统 - 部署指南

## 系统概述

这是一个轻量级的 Flask + SQLite 病人预约管理系统，专为神经心理康复中心设计。

### 系统特点
- ✅ 轻量级部署，无需专门服务器
- ✅ 使用个人 PC 即可运行
- ✅ 支持局域网访问
- ✅ 数据本地存储，安全可靠
- ✅ 操作简单，适合科室自用

## 部署步骤

### 1. 环境准备

#### 检查 Python 安装
打开命令提示符（CMD），输入：
```bash
python --version
```
如果显示 Python 3.7 或更高版本，则环境已就绪。

#### 安装 Flask
在命令提示符中运行：
```bash
pip install flask
```

### 2. 系统部署

#### 方法一：直接运行（推荐）
1. 将整个 `neuro_rehab_app` 文件夹复制到办公室电脑
2. 双击运行 `run.bat` 文件
3. 等待显示服务启动信息
4. 在浏览器中访问 `http://localhost:5000`

#### 方法二：命令行运行
1. 打开命令提示符
2. 进入项目目录：`cd C:\路径\neuro_rehab_app`
3. 运行：`python app.py`
4. 在浏览器中访问 `http://localhost:5000`

### 3. 访问系统

#### 本地访问
- 在办公室电脑上打开浏览器
- 访问：`http://localhost:5000`

#### 局域网访问
- 查看本机 IP 地址：在命令提示符输入 `ipconfig`
- 其他电脑访问：`http://[本机IP]:5000`
- 例如：`http://192.168.1.100:5000`

系统预设了三个用户账号：

| 角色 | 用户名 | 密码 | 说明 |
|------|--------|------|------|
| 预约端 | appointment | appointment123 | 添加、修改、取消预约 |
| 安排端 | schedule | schedule123 | 安排治疗、查看待安排患者 |
| 看板端 | dashboard | dashboard123 | 查看统计、汇总数据 |

## 功能使用说明

### 预约端操作
1. 登录预约端账号
2. 填写患者信息表单（带*为必填项）
3. 点击"添加患者"保存
4. 如需修改，点击"修改"按钮
5. 如需取消，点击"取消"按钮并填写原因

### 安排端操作
1. 登录安排端账号
2. 可选择专业组筛选患者
3. 点击"安排治疗"进行安排
4. 选择是否安排：
   - 是：填写治疗师和时间
   - 否：必须填写原因
5. 黄色高亮表示等待超过1个月的患者

### 看板端操作
1. 登录看板端账号
2. 查看统计概览
3. 查看专业组平均安排时间
4. 查看每月统计汇总
5. 查看患者详细列表

## 数据管理

### 数据存储
- 数据库文件：`neuro_rehab.db`
- 位置：项目目录下
- 格式：SQLite

### 数据备份
1. 关闭系统（关闭运行的窗口）
2. 复制 `neuro_rehab.db` 文件到安全位置
3. 重命名备份文件（如：`backup_20240101.db`）

### 数据恢复
1. 关闭系统
2. 将备份的数据库文件复制到项目目录
3. 重命名为 `neuro_rehab.db`
4. 重新启动系统

## 常见问题

### Q1: 系统无法启动
**解决方法：**
1. 检查 Python 是否已安装
2. 检查 Flask 是否已安装：`pip list | findstr Flask`
3. 检查端口是否被占用：`netstat -ano | findstr :5000`

### Q2: 其他电脑无法访问
**解决方法：**
1. 检查防火墙设置，允许 5000 端口
2. 确保两台电脑在同一局域网
3. 使用本机 IP 地址访问，不要用 localhost

### Q3: 数据丢失怎么办
**解决方法：**
1. 检查是否有备份文件
2. 如无备份，系统会自动创建新数据库
3. 建议定期备份数据库文件

### Q4: 如何修改端口号
**解决方法：**
1. 编辑 `app.py` 文件
2. 找到 `app.run(host='0.0.0.0', port=5000, debug=False)`
3. 修改 `port=5000` 为其他端口（如 8080）
