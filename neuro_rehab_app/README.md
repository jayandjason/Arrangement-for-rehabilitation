# 神经心理康复中心病人预约管理系统

## 系统简介

这是一个轻量级的病人预约管理系统，专为神经心理康复中心设计。系统采用 Flask + SQLite 架构，适合在办公室 PC 上部署运行，无需专门服务器。

## 功能特点

### 1. 预约端功能
- 添加患者信息（姓名、年龄、诊断、患者类型、专业组、主要障碍、就诊类型、预约医生）
- 新增或修改预约时自动提醒重复预约
- 修改患者信息（不能删除数据）
- 取消预约（必须填写取消原因）
- 按预约时间排序显示

### 2. 安排端功能
- 显示所有待安排患者
- 显示预约医生，便于安排端确认预约来源
- 按专业组筛选查看
- 安排治疗（可选择是否安排，未安排需填写原因）
- 设置责任治疗师和联合治疗师
- 黄标提醒（超过1个月未安排）

### 3. 看板端功能
- 查看所有患者列表
- 显示已取消预约及取消原因
- 统计平均安排时间
- 按专业组排序显示
- 每月统计汇总（总预约数、新病人、老病人、取消数）

## 部署步骤

### 1. 环境准备
确保已安装 Python 3.7 或更高版本。

### 2. 安装依赖
在项目目录下运行：
```bash
pip install flask
```

### 3. 启动系统
双击运行 `run.bat` 文件，或在命令行中执行：
```bash
python app.py
```

### 4. 访问系统
在浏览器中访问：`http://localhost:5000`

## 数据库

系统使用 SQLite 数据库，数据文件为 `neuro_rehab.db`，自动创建在项目目录下。

## 注意事项

1. 系统默认监听所有网络接口（0.0.0.0），可在局域网内其他电脑访问
2. 如需修改监听地址或端口，请修改 `app.py` 中的 `app.run()` 参数
3. 系统数据存储在本地 SQLite 数据库，如需备份，请复制 `neuro_rehab.db` 文件
4. 建议定期备份数据库文件以防数据丢失
