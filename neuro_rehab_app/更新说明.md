## 功能详细说明

### 安排端 - 不安排原因验证
选择"不安排"时，系统会强制要求填写原因：
- 如果未填写原因，会显示错误提示："选择不安排时必须填写原因"
- 只有填写原因后才能成功保存

### 安排端 - 已完成安排患者
新增专栏显示已安排治疗的患者：
- 显示患者基本信息和联系方式
- 显示安排时间和治疗师信息
- 支持按专业组筛选

### 看板端 - 时间筛选
- 可设置开始日期和结束日期
- 点击"筛选"按钮应用时间范围
- 点击"清除筛选"返回全部数据

### 看板端 - 数据导出
- 点击"导出数据(CSV)"按钮
- 导出当前筛选条件下的所有数据
- 文件名包含时间范围信息
- 支持Excel打开和编辑

### 看板端 - 数据库备份
- **自动备份**：点击"数据库备份"自动创建备份
- **备份管理**：点击"恢复备份"查看备份列表
- **自动清理**：保留最近10个备份，自动删除旧备份
- **恢复确认**：恢复前会提示确认，防止误操作

## 使用说明

### 使用不安排功能
1. 在安排端点击"安排治疗"按钮
2. 选择"否"（不安排）
3. **必须**填写未安排原因
4. 点击"保存安排"

### 使用时间筛选
1. 在看板端选择开始日期和结束日期
2. 点击"筛选"按钮
3. 查看筛选后的统计数据
4. 可导出筛选后的数据

### 使用数据导出
1. 设置时间范围（可选）
2. 点击"导出数据(CSV)"按钮
3. 下载CSV文件
4. 用Excel打开查看

### 使用数据库备份
1. 点击"数据库备份"按钮
2. 系统自动创建备份
3. 备份文件保存在 `backups/` 目录

### 使用备份恢复
1. 点击"恢复备份"按钮
2. 查看备份列表
3. 选择要恢复的备份
4. 确认恢复操作

## 注意事项

1. **数据安全**：恢复备份会覆盖当前数据，请谨慎操作
2. **备份管理**：系统自动保留最近10个备份
3. **电话格式**：电话字段支持任意格式，如手机号、座机号
4. **时间筛选**：筛选会影响所有统计数据和导出数据
